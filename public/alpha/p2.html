Overview

Your next task will be to analyze a minefield! You'll be given a 100x100 minefield to work with. If a mine explodes, it will cause adjacent mines to explode, too. This can cause a chain reaction, with mines exploding mines exploding mines. The range of a mine's explosion is only the eight tiles surrounding it; in other words, it only has a radius of one.

Input

The input is a 100x100 grid of characters. An 'X' marks a mine, and a space ' ' marks an empty spot.

For this example, a 6x6 minefield will be used instead.

     X
X XX
  X  X
X  X X
 X X
X    X

Output

Mines

You will need to find all the mines that do not cause another mine to explode when detonated. In the example at the right, those mines are highlighted.


Scoring

Each mine has an i (row) position, and a j (column) position. When you find all the mines that fulfill the above requirement, multiply the sum of all i-th positions with the sum of all j-th positions. In other words, add together all i positions into one number and all j positions into another number, then multiply the two together. That will be the score you're looking for.

Given the example, the score is 60.

     X
X XX
  X  X
X  X X
 X X
X    X

     X       // i: 0 | j: 5
X XX         // i: 1 | j: 0
  X  X
X  X X
 X X
X    X       // i: 5 | j: 5
     i             j
(0 + 1 + 5) * (5 + 0 + 5)
       6 * 10 = 60

Constants/Constraints

  • The input will be exactly 100x100 characters large
